DNSMasq and DNSCrypt all in one, with OpenDNS
=====================

# PRE-REQUIREMENT

***Docker-Compose***[^1]

For ubuntu & Mac OSX

```bash
sudo pip -H install --upgrade docker-compose
```

# INSTALL

```bash
git clone https://github.com/thiswind/dnscrypt_dnsmasq.git

cd dnscrypt_dnsmasq

docker-compose up
```
